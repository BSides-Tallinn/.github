# BSides Tallinn

**Community-driven information security event in Tallinn, Estonia.**

[BSides](https://bsides.org/) is a worldwide grassroots infosec event format organized by local communities. BSides Tallinn has been bringing together security professionals, researchers, hackers, students and enthusiasts since 2021.

## What we do

- Organize the annual **BSides Tallinn** conference — talks, workshops, villages, and networking
- All content is proposed and selected by the community through an open call for papers
- Foster knowledge sharing across public and private sector, academia, and the broader infosec community
